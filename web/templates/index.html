About Project

  • The main concept of this project was to create a manager for an ecommerce company with a basic domain.
  • This page is a prototype web app that presents useful information about the tables and lets users interact with the database backend.
  • We had to implement the Integrity Constraints in sql with Stored Procedures/Triggers, develop advanced queries, implement a view for the products sales of the company, develop OLAP queries and speed up queries by creating indexes on certain columns.
